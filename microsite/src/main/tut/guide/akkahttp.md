sup has modules for akka-http and circe (although you can use any JSON library compatible with akka-http):

```tut:passthrough
sup.microsite.sbtDependencies("akka-http", "circe")
```

Imports:
```tut:silent
import sup._
import sup.modules.akkahttp._, sup.modules.circe._
```
